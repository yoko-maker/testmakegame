# 🎮 ミニゲームアーケード

Streamlit で作った、6つのミニゲームが遊べる1つの作品です。
共通の **コイン経済**・**プレイヤープロフィール**・**実績システム** で全ゲームがつながっています。

## 収録ゲーム

| ゲーム | 内容 | 報酬 |
|---|---|---|
| 🔢 数当てゲーム | 1〜100の隠し数字を当てる | 少ない回数ほど高コイン（最大55） |
| ✊ じゃんけんバトル | CPUと対戦、5勝でボーナス | 1勝 +8／5勝 +30 |
| 🧠 神経衰弱 | 4×4の絵文字メモリーマッチ | 少ない手数ほど高コイン |
| ❓ クイズ | 全8問の4択クイズ | 正解 +5／全問正解 +30 |
| 🎰 スロットマシン | 3リールスロット（1回 -10） | 3つ揃いで最大 +100 |
| 🔤 ハングマン | 単語当て（英語／日本語ひらがな対応） | クリア +25 |

## つながる仕組み

- **共通ウォレット**: どのゲームで稼いだコインも1つの財布に貯まり、スロットの軍資金などに使えます。
- **レベル**: 累計獲得コイン100枚ごとに1レベルアップ。
- **実績**: 「千里眼」「じゃんけん王」「大富豪」などをホーム画面で収集。

## 物語と隠し要素

- **店主「マスター」**: ホーム画面に登場。累計獲得コインのしきい値ごとに身の上話が
  段階解放され、最後に「なぜ古い筐体がここに並ぶのか」という真相にたどり着きます。
- **景品交換所**: 貯めたコインで *マスターの追加エピソード* / *収集バッジ* /
  **7本目の隠しゲーム「数列の記憶」** を解放できます。
- **フレーバー**: ハングマンの単語やクイズ、スロットにさりげない小ネタ（ECHO / NOXA / 404）を散りばめています。

> このアーケードは、6作品が背景でゆるく繋がる共通世界観（ノクサ研究機構・プロジェクト ECHO）の入口でもあります。

## 動かし方

```bash
pip install -r requirements.txt
streamlit run app.py
```

ブラウザで `http://localhost:8501` が開きます。

## 構成

- `app.py` — アプリ本体（共通システム＋6ゲーム）
- `requirements.txt` — 依存パッケージ

> ⚠️ コイン・レベル・実績はセッション中のみ保持されます（ブラウザを閉じるとリセット）。
> 永続化したい場合は今後の拡張で対応できます。
